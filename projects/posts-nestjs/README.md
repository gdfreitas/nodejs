# Nest.js Project

## Instalando CLI

- `npm i -g @nestjs/cli`

## Instalando dependências

```bash
$ npm install
```

## Rodando servidor

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Executando testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
